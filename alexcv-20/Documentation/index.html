Alex - CV / Resume / Portfolio is Html5 + CSS3 responsive template and works fine in all latest browsers. The code is clearly written and you will find comments for each section parts.

Structure of template files

You will need some knowledge 'html/css' for understand the code below.
You can use special text editors like 'Notepad++', 'Sublime Text' to edit the code.

Structure of files

  • index.html All Pages - HTML files that is located in the root of the site
  • portfolio Directory contain all portfolio images
  • css Directory contain all stylesheet files
  • js Directory contain all javascript files
  • images Directory contain logo and all images

Installation

Follow the steps below to get started with your Site:

  • Open the HTML Folder to find all the files
  • You will need to upload these files to your Web Server using FTP or Cpanel in order to use it as your Website.
  • Make sure you upload all the files/folders listed below:
    - index.html
    - portfolio
    - js
    - img
    - css
  • You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

Page Structure
